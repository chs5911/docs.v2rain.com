# V2RayN

## 简介

### V2RayN 是 PC 平台上一个对新手比较友好的 V2Ray Windows 客户端。

## 系统要求

### Windows7 32bit 及以上 ，需要 Microsoft .NET Framework 4.6 或更高版本。

## 如何使用

1. [点击下载 V2RayN](https://yun.nonage.me/s/aotvng82)
