# Clash For Android
## 简介

#### Clash For Android 是一款 Android 平台的网络代理软件，可以同时支持 SS 和 Vmess 协议。
- 开发人员：[Kr328](https://github.com/Kr328)
- 项目地址：[Github](https://github.com/Kr328/ClashForAndroid)
- 下载链接：[ClashForAndroid](https://yun.nonage.me/s/7plknb7i)

##  系统要求
- Android 7.0 及以上
- 64 位 ROM (armeabi-v7a , arm64-v8a, x86 or x86_64 Architecture)

## 如何使用
#### 1. 下载应用[点击下载](https://yun.nonage.me/s/7plknb7i)
#### 2. 安装完成后打开应用
![cfa01](https://i.loli.net/2020/04/05/Btro2XyjFRnHJhG.png)
#### 3. 点击配置，添加配置
![cfa02](https://i.loli.net/2020/04/05/XvRCnlth1mboDSq.png)

**创建配置选择从 URL 导入**

#### 4. 打开官网复制订阅链接
1. 手机浏览器打开[官网](http://飞机.tech/),并登陆到用户中心。
![cfa03](https://i.loli.net/2020/04/05/ZgEk1l5UDrK9Jes.png)
2. 打开产品页面复制您的**Clash**订阅链接。
![cfa04](https://i.loli.net/2020/04/05/V5svtEPgKOXT8h3.png)
3. 复制成功
![cfa05](https://i.loli.net/2020/04/05/czK2NRai51heLZy.png)

#### 5.返回 Clash For Android 添加配置 URL

- 配置**名称**随意，可以填**AcmeCloud**, URL 填官网产品页面复制的**Clash订阅链接**,**自动更新**按自己需求填（可不填）。
![cfa06](https://i.loli.net/2020/04/05/mqMaRnF8YZyOhor.png)
- 点击右上角保存配置，并选中你导入的配置
